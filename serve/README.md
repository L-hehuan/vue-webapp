# 项目创建

---
1. 全局安装脚手架
```bash
cnpm install express-generator -g
```
2. 使用ejs模板引擎来创建node项目
```bash
express --view=ejs node-fullapi
cd node-fullapi
cnpm install
npm start
```
3. 默认运行：http://localhost:3000

4. 数据库
```bash
cnpm install mongoose -S
```
